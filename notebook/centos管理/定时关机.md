### 定时关机

```bash
sudo shutdown -h 23:00 //在23点定时关机
```

### 延时关机

```bash
sudo shutdown -h +120 //两小时后关机
```

### 取消前一个关机命令

```bash
sudo shutdown -c
```

### 关机重启

```bash
sudo shutdown -r now 
```

### 关机后不重启

```bash
sudo shutdown -h now //立即关机
```

### 定时重启

```bash
sudo shutdown -r 11:00 //11点重启
```

### 延时重启

```bash
sudo shutdown -r +11 //11分钟后重启
```
